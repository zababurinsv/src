Pets.register(pet)

Registers a new pet into the Pet Adoption Center.

Kind: static method of Pets

Param Type Description
pet Pet The pet.

Example

const sdk = require('pet-adoption-center');

const transaction = sdk.pets.register({
    name: 'Atticus',
    type: sdk.PET_TYPES.DOG,
    breed: sdk.DOG_BREEDS.FRENCH_BULLDOG,
    dateOfBirth: '2010-03-10',
    neutered: false
});

Pets.adopt(options) ⇒ TransactionResponse

Some lucky pet is getting adopted!

Kind: static method of Pets
Returns: TransactionResponse - Details about the transaction.

Param Type Description
options object Options object.
options.pet Pet The pet being adopted.
options.supporter Supporter The supported who adopts the pet.

Example

const sdk = require('pet-adoption-center');

// Register a pet
const registerTx = await sdk.pets.register({
    name: 'Atticus',
    type: sdk.PET_TYPES.DOG,
    breed: sdk.DOG_BREEDS.FRENCH_BULLDOG,
    dateOfBirth: '2010-03-10',
    neutered: false
});

// Enroll a supporter
const enrollTx = await sdk.supporters.enroll({
    name: 'Atticus',
    type: sdk.PET_TYPES.DOG,
    breed: sdk.DOG_BREEDS.FRENCH_BULLDOG,
    dateOfBirth: '2010-03-10',
    neutered: false
});

// It's a match!
const adoptTx = sdk.pets.adopt({ pet: registerTx.data, supporter: enrollTx.data });

Supporters.enroll(supporter) ⇒ TransactionResponse

Enrolls a new supporter into our Pet Adoption Center database.

Kind: static method of Supporters
Returns: TransactionResponse - Details about the transaction.

Param Type Description
supporter Supporter The supporter.

Example

const sdk = require('pet-adoption-center');

const transaction = sdk.supporters.enroll({
    firstName: 'Atticus',
    lastName: sdk.PET_TYPES.DOG,
    dateOfBirth: '1983-01-01'
});
